# Domain 5 – Security Governance, Risk, and Compliance

This domain focuses on governance, risk management, compliance, third-party risk, audits, and security awareness.

---

## 5.1 Security Governance

Security governance defines how an organization manages security policies, procedures, and decision-making.

---

### Governance Components

#### Guidelines
- Recommended best practices  
- Not mandatory  

#### Policies
Formal rules defining security requirements.

**Examples:**
- Acceptable Use Policy (AUP)  
- Information Security Policy  
- Business Continuity Policy  
- Disaster Recovery Policy  
- Incident Response Policy  
- Change Management Policy  

---

### Software Development Lifecycle (SDLC)

Secure development lifecycle stages:

1. Planning  
2. Development  
3. Testing  
4. Deployment  
5. Maintenance  

---

### Standards

Mandatory technical requirements supporting policies.

**Examples:**
- Password standards  
- Access control standards  
- Encryption standards  
- Physical security standards  

---

### Procedures

Step-by-step instructions.

**Examples:**
- Onboarding/offboarding  
- Change management  
- Security playbooks  

---

### External Considerations

Organizations must comply with:

- Regulatory requirements  
- Legal obligations  
- Industry standards  
- Regional and international laws  

**Examples:**
- GDPR  
- HIPAA  
- PCI DSS  

---

### Governance Structures

| Structure        | Description |
|------------------|------------|
| Boards           | Strategic leadership |
| Committees       | Manage initiatives |
| Government Bodies| Regulatory enforcement |

---

### Governance Models

| Model         | Description |
|---------------|------------|
| Centralized   | Single authority |
| Decentralized | Distributed control |

---

### Roles and Responsibilities

| Role        | Description |
|-------------|------------|
| Owner       | Responsible for data/system |
| Controller  | Determines data processing |
| Processor   | Processes data |
| Custodian   | Maintains and protects data |
